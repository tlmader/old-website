Theodoric

v0.1

A 2D top-down action RPG built on the Phaser HTML5 game framework

Author: Ted Mader

Genre: Action RPG

Tools Used:
* Phaser - Game engine
* Wamp - Local testing
* Github Pages - Remote testing

Game Client:
Download the package, unzip, and run Theodoric.exe. WARNING: The package has an issue with playing the game’s audio - the audio may sound sluggish.

Controls:
* WASD - Move
* Left Mouse Button - Hold down to attack
* Spacebar - Cast spell

Objectives:
* Gather gold and slay monsters to achieve the highest score possible!
